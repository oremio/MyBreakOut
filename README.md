启动方式：点击 MyOpenGLBreakOut\MyOpenGLBreakOut.exe 。如果报找不到 MSVCP140D.dll等等，那就去网上下，或者装个Visual Studio。

游玩方式：

- W/S 选择关卡
- Enter启动
- A/D 移动平台
- 空格弹可可
- 平台吃到道具，会获得 buff 或 debuff
